# 云仓库助手客户端 Mod

这是一个 Minecraft 1.21.11 Fabric 客户端 mod。它不需要服务器安装，只是在客户端提供简洁 GUI，帮玩家快速向云仓库机器人发送私聊命令。

## 功能

- 默认按 `Y` 打开界面，可在游戏按键设置中修改。
- 操作页只保留常用操作：
  - 个人存入
  - 个人查询
  - 个人取出
  - 组织存入
  - 组织查询
  - 组织取出
- 取出输入框支持一次填写多个物品，例如 `石头64 红石块3 451038 1`。
- 设置页可以修改机器人名字，默认 `NanKinz1`。
- 设置页可以添加/删除本地组织仓库名，用于快速发送 `组织名 存` 和 `组织名 取 ...`。

这个 mod 面向普通玩家使用，不提供机器人管理员命令，也不提供组织成员管理、额度设置、库存修改、仓库扫描等入口。

## 构建

需要 Java 21。进入本目录后运行：

```powershell
.\gradlew.bat build
```

构建完成后，把 `build/libs/cloud-store-mod-0.1.0.jar` 放进客户端 `mods` 目录，同时安装 Fabric Loader 和 Fabric API。
