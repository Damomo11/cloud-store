# 云仓库机器人命令说明

公屏使用命令时必须以 `!` 开头，例如 `!查 石头`。

私聊机器人时可以省略 `!`，例如私聊发送 `查 石头`、`存`、`取 石头64` 也可以。

机器人一次只服务一名玩家。如果机器人正在服务别人，会提示“机器人忙，请稍后再试”。

## Web 库存面板

可以直接用网页上的临时验证码登录；如果想固定密钥，先在游戏内私聊机器人设置登录密钥：

```text
设置密钥 1234
```

之后打开 Web 面板，在首页输入这个密钥登录。每次重新设置都会覆盖旧密钥。

登录后只能看到自己的个人仓库，以及自己加入的组织仓库。组织仓库成员看不到其他无关组织仓库。

Web 面板左侧有“库存”和“交易记录”。库存页可以点“远程存储”或“取货”，实际仍由机器人在游戏内执行，玩家需要接受机器人的传送请求。

## 个人仓库

个人仓库只属于你自己，其他玩家不能取你的个人库存。

### 查询库存

```text
!查 物品名
!查询 物品名
```

示例：

```text
!查 石头
!查 经验修补一
!查 烟花火箭三
```

查询一次只能查一种物品，避免刷屏。

### 存物品

```text
!存
!存物品
```

流程：

1. 发送 `!存`
2. 机器人会向你发送传送请求
3. 接受传送后，把要存的物品丢给机器人
4. 等待 10 秒，或发送 `!完成` / `!结束` 提前结束
5. 机器人回仓库入库，并把物品记到你的个人仓库

提前结束：

```text
!完成
!结束
```

### 取物品

```text
!取 物品名数量
!取 物品名 数量
!取物品 物品名数量
```

示例：

```text
!取 石头64
!取 石头 64
!取 石英块64 石头128 红石块3
!取 经验修补一 1
!取 烟花火箭三 64
!取 451038 1
```

如果库存不够，机器人会告诉你当前只有多少，并按最大可取数量出货。

复杂附魔物品、改名物品、潜影盒等会显示 6 位短码，例如 `451038`。取这类物品时可以直接用短码。

### 查看额度

```text
!额度
```

会显示你当前已使用多少格库存额度。

## 组织仓库

组织仓库是多人共用仓库，和个人仓库完全分开。组织仓库名全服唯一，不能包含空格，最多 15 个字。

每个玩家最多创建 3 个组织仓库。创建者自动成为该组织仓库管理员。

### 创建和查看

```text
!创建仓库 仓库名
!我的仓库
```

示例：

```text
!创建仓库 椰汁橙
!我的仓库
```

### 查询组织仓库库存

```text
!仓库名 查 物品名
```

示例：

```text
!椰汁橙 查 石头
!椰汁橙 查 经验修补一
!椰汁橙 查 烟花火箭三
```

### 存入组织仓库

```text
!仓库名 存
```

示例：

```text
!椰汁橙 存
```

流程和个人仓库存物品一样，但物品会记到这个组织仓库。

### 从组织仓库取物品

```text
!仓库名 取 物品名数量
!仓库名 取 物品名 数量
```

示例：

```text
!椰汁橙 取 石头64
!椰汁橙 取 石英块64 红石块3
!椰汁橙 取 经验修补一 1
!椰汁橙 取 烟花火箭三 64
!椰汁橙 取 451038 1
```

只有组织仓库成员可以查询、存入、取出该组织仓库的物品。

### 成员和权限

组织仓库管理员可以查看成员列表：

```text
!仓库名 仓库成员
!仓库名 成员
!仓库名 查看成员
!仓库名 成员列表
```

示例：

```text
!椰汁橙 成员
```

组织仓库管理员可以添加或删除成员：

```text
!仓库名 添加成员 玩家ID
!仓库名 删除成员 玩家ID
```

也可以使用短命令：

```text
!仓库名 加成员 玩家ID
!仓库名 删成员 玩家ID
```

组织仓库管理员可以管理组织仓库管理员：

```text
!仓库名 添加管理员 玩家ID
!仓库名 删除管理员 玩家ID
```

也可以使用这些别名：

```text
!仓库名 添加管理 玩家ID
!仓库名 加管理员 玩家ID
!仓库名 加管理 玩家ID
!仓库名 删除管理 玩家ID
!仓库名 删管理员 玩家ID
!仓库名 删管理 玩家ID
```

仓库创建者不能被删除成员，也不能被取消管理员权限。

## 管理员命令

以下命令只有机器人管理员可以使用。

### 同步仓库

```text
!同步
!同步 半径
!同步仓库
!同步仓库 半径
```

示例：

```text
!同步 16
!同步仓库 30
```

同步会扫描附近木桶，更新机器人记录的木桶位置和槽位内容。多出来、无法确认归属的物品会记入 `momo`。

### 清空测试库存

```text
!清空库存 确认
!清理库存 确认
```

这个命令会清空库存账本、木桶位置、木桶槽位、存取记录和历史物品档案。

不会清空配置别名、物品人工命名规则、机器人管理员、玩家、额度、自定义仓库、仓库成员。

### 查询 momo

```text
!查 momo 物品名
!查询 momo 物品名
```

示例：

```text
!查 momo 石头
```

`momo` 是未知归属和缓冲库存账户。

### 查询玩家额度

```text
!额度 玩家ID
!额度 仓库:仓库名
```

示例：

```text
!额度 PlayerName
!额度 仓库:椰汁橙
```

### 设置玩家额度

```text
!设置额度 玩家ID 格数
!设置额度 仓库:仓库名 格数
```

示例：

```text
!设置额度 PlayerName 270
!设置额度 仓库:椰汁橙 270
```

额度单位是“格”。270 格约等于 17280 个普通 64 堆叠物品。

### 修改玩家库存账本

```text
!增加 玩家ID 物品名 数量
!减少 玩家ID 物品名 数量
```

示例：

```text
!增加 PlayerName 石头 64
!减少 PlayerName 绿宝石 10
```

这个命令只修改账本，不会移动木桶里的实体物品。

### 添加机器人管理员

```text
!加管理员 玩家ID
!加管理 玩家ID
```

示例：

```text
!加管理员 PlayerName
```

只有现有机器人管理员可以添加新的机器人管理员。
